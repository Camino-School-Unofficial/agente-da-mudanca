Cartas na mesa

Nossa interface funciona como várias ficha sobre a mesa. Cada uma delas possui uma parte do conteúdo. Mas é o conjunto que faz toda a diferença ;)
Para navegar entre elas você pode usar as setas do teclado. Ou se preferir, pode usar o mouse nos botões no canto direito.

Pontos de vista

Pressiosse ESC para visualizar todas as fichas.
Você pode navegar entre as fichas usando as setas do seu teclado.
Pressione ESC novamente ou clique para voltar a visão completa da ficha.

Zoom

Segure alt e clique em algum elemento para fazer um zoom nele. Clique novamente para sair do zoom.

Barra de progressão

Uma pequena barra ao fim da ficha apresenta visualmente a progressão do conteúdo.
Hora de acessar o próximo tópico na coluna lateral. Vamos lá?

Exporte

Você quer seu material offline para imprimir ou tomar notas? Você pode exportar para PDF
Depois de clicar no link é só selecionar tudo e pedir para seu navegador imprimir. Assin, você pode salvar o .pdf gerado :)
